# Deep Learning Study

深度学习学习项目，包含论文复现、模型实现和实战练习。

## 目录结构

```
deep-learning/
├── papers/          ← 论文复现代码
├── models/          ← 模型实现
├── exercises/       ← 练习代码
├── projects/        ← 实战项目
└── notes/           ← 学习笔记
```

## 环境

- Python 3.10+
- PyTorch 2.0+
- CUDA (如有 GPU)

## 学习路径

1. PyTorch 基础
2. CNN 实现（AlexNet, VGGNet, ResNet）
3. Transformer 实现
4. 预训练模型使用（BERT, GPT）
5. 微调技术（LoRA, QLoRA）
6. 多模态模型（CLIP, LLaVA）
